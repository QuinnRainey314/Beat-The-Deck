Beat the Deck - Rules

Beat the Deck is a single-player card game where you try to use up all the cards in your draw pile by correctly guessing whether the next card will be higher or lower than the top card of a chosen stack.

1. Setup: The game starts with a standard 52-card deck. Nine cards are dealt face-up into a 3x3 grid (or a different number of stacks if you change the slider). The remaining cards form the draw pile.

2. Gameplay:

  • Select one of the face-up card stacks.
  • Bet whether the next card drawn from the draw pile will be higher or lower in value than the top card of the selected stack. (Ace is high).
  • If you guess correctly, the drawn card is placed on top of the stack, and you can choose another stack (or the same one) to continue.
  • If you guess incorrectly, the stack is "flipped" (grayed out) and can no longer be used.

3. Winning and Losing:

  • You win if you use up all the cards in the draw pile before all the stacks are flipped.
  • You lose if all the stacks are flipped before you use up all the cards in the draw pile.

4. Changing the Number of Stacks

  • You can adjust the number of starting stacks on the title screen by moving the slider, which can make the game easier or more challenging.
